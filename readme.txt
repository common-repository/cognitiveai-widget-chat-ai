== Description ==

Этот плагин использует сторонний сервис CognitiveAI для работы плагина чата с нейросетями. При использовании этого плагина данные могут передаваться на серверы CognitiveAI. Пожалуйста, ознакомьтесь с [условиями использования](https://cognitiveai.ru/polzovatelskoe-soglashenie/) и [политикой конфиденциальности](https://cognitiveai.ru/polzovatelskoe-soglashenie/) CognitiveAI, чтобы узнать больше о том, как обрабатываются ваши данные.

Плагин "Чат с нейронными сетями" представляет собой превосходный инструмент для создания и редактирования контента и генерации изображений для вашего сайта. Упростите задачи контент-менеджера, ускорьте процесс наполнения товаров, создания блог-постов или ответа на любые вопросы.

Плагин чата с нейросетями - это интеграция виджета от [CognitiveAI](https://cognitiveai.ru/vidzhet-s-nejrosetyami-dlya-sajta/). Cognitiveai widget chat AI доступен при администрировании сайта на Wordpress. Чат имеет доступ к нейросетям GPT-3.5, GPT-4 и DALL-E от OpenAI, доступ к которым предоставляется напрямую через виджет без необходимости регистрации на OpenAI, YandexGPT и другим моделям, подключенным к платформе CognitiveAI.

После установки плагина не забудьте перейти в его настройки – "Настройки\Cognitiveai widget chat AI".

После настройки плагина вы увидите в правом нижнем углу иконку чат-виджета, кликнув на которую откроется окно чата с нейросетями. Данный чат будет доступен для авторизованных пользователей с правами администратора на всех страницах сайта, включая публичную часть.

Внимание!
Платформа CognitiveAI является платной. При регистрации вам будет доступно 10 руб. для тестирования. Далее, чтобы использовать нейронные сети, необходимо пополнить баланс. С вашего счета будут списываться средства за использованные токены и сгенерированные изображения. Вы также можете подключить подписку, которая позволит экономить на использовании нейросетей, а также даст возможность использовать ваши интеграционные данные. Например, вы сможете вставить свой API-ключ от OpenAI на данной платформе и оплачивать услуги OpenAI напрямую. Эта возможность доступна при оформлении подписки.

Если вам понравился этот плагин, то, пожалуйста, поставьте ему 5 звезд.

== Installation ==
= Инструкция по установке =

1. Установите плагин через меню "<strong>Плагины\Добавить новый</strong>" (искать "Cognitiveai widget chat AI").
2. Активируйте плагин в меню "Плагины".
3. Чтобы плагин работал, вам необходимо указать ID Виджета и включить его. Чтобы получить ID Виджета, вам необходимо зарегистрироваться на платформе CognitiveAI по [ссылке](https://lk.cognitiveai.ru) и после регистрации на [странице](https://lk.cognitiveai.ru/chat-widget) создать виджет для вашего домена, где вы увидите ID Виджета. 
4. Зайдите в настройки плагина "<strong>Настройки\Cognitiveai widget chat AI</strong>", укажите ID Виджета, поставьте галочку и сохраните.
5. Это все.
